## Project Description
A Flutter app where users report and receive 
real-time alerts about safety incidents in their 
neighborhood, with an SOS button, live incident 
map, anonymous reporting, and offline support.

**Screens:** Map, Feed, Report Incident, 
Incident Detail, SOS, Notifications, 
Community Groups, Analytics, Settings, Admin

**Tools:** Flutter, Firebase Auth, Firestore, 
BLoC, WebSockets, Google Maps, Hive

## Other Information
- Inspired by Citizen app
- Targets Romanian neighborhoods (112 emergency)
- Anonymous reporting for sensitive incidents
